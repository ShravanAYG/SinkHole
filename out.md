source](https://wiki.freecad.org/index.php?title=Main_Page&action=edit&section=1 "Edit section's source code: Introduction")\]\n\nThis is the documentation wiki of [FreeCAD](https://freecad.org/). There are several ways to use this documentation: by exploring the hubs, by following the manual, or by using the Help menu entries from within FreeCAD. This is a work in progress, written by the community of users and developers of FreeCAD. If you find information that is wrong or missing, please [help FreeCAD](https://wiki.freecad.org/Special:MyLanguage/help_FreeCAD "Special:MyLanguage/help FreeCAD")!\n\n## The hubs\[ [edit](https://wiki.freecad.org/index.php?title=Main_Page&veaction=edit&section=2 "Edit section: The hubs") \| [edit source](https://wiki.freecad.org/index.php?title=Main_Page&action=edit&section=2 "Edit section's source code: The hubs")\]\n\n[![](https://wiki.freecad.org/images/thumb/9/9b/Crystal_Clear_app_display.png/64px-Crystal_Clear_app_display.png)](https://wiki.freecad.org/index.php?title=File:Crystal_Clear_app_display.png&filetimestamp=20090202153417&)\n\n[Users hub](https://wiki.freecad.org/User_hub "User hub"): This page contains documentation useful for FreeCAD users in general: a list of all the workbenches, detailed instructions on how to install and use the FreeCAD application, tutorials, and all you need to get started.\n\n[![](https://wiki.freecad.org/images/thumb/1/1a/Crystal_Clear_app_terminal.png/64px-Crystal_Clear_app_terminal.png)](https://wiki.freecad.org/index.php?title=File:Crystal_Clear_app_terminal.png&filetimestamp=20090202153428&)\n\n[Power users hub](https://wiki.freecad.org/Power_users_hub "Power users hub"): This page gathers documentation for advanced users and people interested in writing Python scripts. There you will also find a repository of macros, instructions on how to install and use them, and more information about customizing FreeCAD to your specific needs.\n\n[![](https://wiki.freecad.org/images/thumb/0/0b/Crystal_Clear_app_tutorials.png/64px-Crystal_Clear_app_tutorials.png)](https://wiki.freecad.org/index.php?title=File:Crystal_Clear_app_tutorials.png&filetimestamp=20090202153438&)\n\n[Developers hub](https://wiki.freecad.org/Developer_hub "Developer hub"): This section contains material for developers: How to compile FreeCAD yourself, how the FreeCAD source code is structured, how to navigate the source code, how to develop new workbenches, and embed FreeCAD in your own application. "Online Help Toc/uk") |\n\n## Get involved\[ [edit](https://wiki.freecad.org/index.php?title=Main_Page&veaction=edit&section=5 "Edit section: Get involved") \| [edit source](https://wiki.freecad.org/index.php?title=Main_Page&action=edit&section=5 "Edit section's source code: Get involved")\]\n\n### How to participate\[ [edit](https://wiki.freecad.org/index.php?title=Main_Page&veaction=edit&section=6 "Edit section: How to participate") \| [edit source](https://wiki.freecad.org/index.php?title=Main_Page&action=edit&section=6 "Edit section's source code: How to participate")\]\n\nThere is plenty to do inside the FreeCAD project, if you are interested in helping us. Of course, there are programming tasks for C++ or Python programmers, but there are also many things you can do even if you cannot code, such as\n\n- writing documentation and [editing the wiki (en)](https://wiki.freecad.org/WikiPages "WikiPages")\n- helping newcomers\n- translating the application and documentation\n- helping with the packaging of the latest release of FreeCAD for your favourite operating system\n- helping other people around you to discover FreeCAD.\n\nThe [help FreeCAD](https://wiki.freecad.org/Special:MyLanguage/help_FreeCAD "Special:MyLanguage/help FreeCAD") page describes it all with more details. Starting from 2016, FreeCAD also participates in the [Google Summer of Code](https://wiki.freecad.org/Google_Summer_of_Code "Google Summer of Code").\n\n### Source code\[ [edit](https://wiki.freecad.org/index.php?title=Main_Page&veaction=edit&section=7 "Edit section: Source code") \| [edit source](https://wiki.freecad.org/index.php?title=Main_Page&action=edit&section=7 "Edit section's source code: Source code")\]\n\nFreeCAD can be compiled on all platforms using [CMake](https://cmake.org/). The source code is [LGPL](https://en.wikipedia.org/wiki/GNU_Lesser_General_Public_License)-licensed and hosted on [GitHub](https://github.com/FreeCAD/FreeCAD) and mirrored on [GitLab](https://gitlab.com/freecad/FreeCAD), [SourceForge](https://sourceforge.net/projects/free-cad/) and [CodeBerg](https://codeberg.org/FreeCAD/FreeCAD). There are build instructions for [Windows](https://wiki.freecad.org/Compile_on_Windows "Compile on Windows"), [Linux](https://wiki.freecad.org/Compile_on_Linux "Compile on Linux") and [MacOS](https://wiki.freecad.org/Compile_on_MacOS "Compile on MacOS"). The source code documentation is [hosted here](https://freecad.org/api/), generated by [Doxygen](https://wiki.freecad.org/Doxygen "Doxygen"), and [documented on the wiki](https://wiki.freecad.org/Source_documentation "Source documentation").\n\n### About the development\[ [edit](https://wiki.freecad.org/index.php?title=Main_Page&veaction=edit&section=8 "Edit section: About the development") \| [edit source](https://wiki.freecad.org/index.php?title=Main_Page&action=edit&section=8 "Edit section's source code: About the development")\]\n\nCheck the [Development roadmap](https://wiki.freecad.org/Development_roadmap "Development roadmap") for news about what is being planned, the [Changelog](https://tracker.freecad.org/changelog_page.php) and [Roadmap](https://tracker.freecad.org/roadmap_page.php) pages on the [FreeCAD tracker](https://github.com/FreeCAD/FreeCAD/issues) to see the progress towards next release, or the [Project statistics](http://www.ohloh.net/p/freecad) for even more information about the FreeCAD codebase. All the development communication happens on the [forum](https://forum.freecad.org/), so be sure to visit it if you are interested in
